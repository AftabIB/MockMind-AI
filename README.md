# MockMind AI — Adaptive Mock Test Platform

> AI-powered adaptive mock test platform that generates theory & quiz questions using **Google Gemma 3 27B** via Google AI Studio. Built with Next.js.

## ✨ Features

### Core
- **AI-Generated Theory** — Level-appropriate theory content (800–1200 words) generated by Gemma 3 27B with code examples
- **Adaptive Quiz System** — 5 questions per level where difficulty adapts based on your answers in real-time
- **3 Progressive Levels** — Beginner → Intermediate → Advanced, each with its own theory + quiz cycle
- **15+ Popular Topics** — Python, JavaScript, React, TypeScript, Node.js, SQL, Java, C++, Go, Rust, Docker, AWS, ML, Next.js, MongoDB — or type any custom topic
- **Instant Feedback** — See explanations for every answer immediately after submission
- **Final Scorecard** — Overall grade (S/A/B/C/D), per-level breakdown, star ratings, and performance feedback

### Adaptive Question System

- **Decision Tree Logic** — Correct answers lead to trickier variants, wrong answers lead to simpler variants
- **Brain-Pressure Questions** — Medium and hard questions combine multiple concepts, test edge cases, and require deep thinking (interview-level difficulty)
- **Question Deduplication** — Normalizes and compares questions to prevent repeats across levels

### Optimization Techniques
- **AbortController** — Prevents React Strict Mode from doubling API calls in development
- **Token Budget System** — Per-session token tracking with auto-cleanup to stay within tier limits
- **Singleton AI Client** — Single `GoogleGenAI` instance reused across all requests

### UI/UX
- **Dark Glassmorphism Design** — Premium dark theme with glass surfaces, glow effects, and animated gradients
- **Animated Borders** — CSS mask technique for smooth gradient border animations
- **Responsive** — Full responsive design with breakpoints at 480px, 768px, 1024px
- **Touch Optimized** — Disabled hover effects on touch devices, 48px minimum touch targets
- **Safe Area Support** — Proper rendering on notched/modern mobile devices
- **Shimmer Loading** — Animated skeleton loaders during AI generation

## 🚀 Getting Started

### 1. Get a free Gemini API key

Visit [Google AI Studio](https://aistudio.google.com/app/apikey) and create a free API key.

### 2. Set up Upstash Redis (for visitor counter)

1. Go to [console.upstash.com](https://console.upstash.com) → Create a free account
2. Create a new Redis database (free tier: 500K commands/month)
3. Copy the **REST URL** and **REST Token** from the dashboard

### 3. Clone & set up environment

Create a `.env.local` file in the root directory:

```
GEMINI_API_KEY=your_api_key_here
UPSTASH_REDIS_REST_URL=your_upstash_rest_url_here
UPSTASH_REDIS_REST_TOKEN=your_upstash_rest_token_here
```

### 4. Install & run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
