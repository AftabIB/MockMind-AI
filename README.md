# MockMind AI — Adaptive Mock Test Platform

An AI-powered adaptive mock test platform that generates theory content and quiz questions using **Google Gemma 3 27B** via Google AI Studio.

## Features

- **AI-Generated Theory** — Comprehensive, level-appropriate theory content generated by Gemma 3 27B
- **Adaptive Quiz System** — 5 questions per level with a mix of theory and coding questions
- **3 Difficulty Levels** — Beginner → Intermediate → Advanced
- **Smart Question Generation** — Questions are based on the theory you just studied, never repeated
- **Instant Feedback** — See explanations for every answer
- **Final Scorecard** — Overall grade, per-level breakdown, and performance feedback

## Tech Stack

- **Frontend:** Next.js 14, React 18, Tailwind CSS
- **AI Model:** Google Gemma 3 27B IT (via Google AI Studio)
- **SDK:** `@google/genai`

## Getting Started

### 1. Get a free API key

Visit [Google AI Studio](https://aistudio.google.com/app/apikey) and create a free API key.

### 2. Set up environment

```bash
cp .env.local.example .env.local
```

Edit `.env.local` and paste your API key:

```
GEMINI_API_KEY=your_api_key_here
```

### 3. Install & run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## API Usage

Per session (3 levels): **6 API calls total**
- 3 theory generation calls
- 3 batch question generation calls (5 questions each)
- ~15,000 tokens total per session
